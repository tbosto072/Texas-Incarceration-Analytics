# Texas Incarceration Analytics
An exploratory data analysis project that explores inmate demographics, disparities in sentencing, county trends, and parole insights from a dataset that includes many currently incarcerated Texans across multiple facilities.

## Objectives
The objective of this project is to analyze inmate demographics trends, identify any negative patterns in sentencing, and see where any resources can be placed for improvement in problematic counties.<br>Statistics such as following will be explored: <br>
- Male vs Female Ratio
- Average age of inmates
- Most common offense
- Racial disparity between sentencing lengths
- County where most inmates come from

## Problem Statement
The goal of this project is to see what counties are the most troubled, and to identify any disparities between sentencing length of different races to identify racial bias/discrimination. By identifying these metrics, I can turn that into actionable insight and recommend resources to help keep crime lower in these counties and prevent less people from getting jailed. 

## Tools Used
The tools I will be using for this project will be Jupyter Notebook, Python, SQL, DataGrip, Pandas, and Power BI.
